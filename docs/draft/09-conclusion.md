# Conclusion — IS EdTech Main Paper

**Word count target:** ~500 words  
**Structure:** Summary of contributions → Limitations → Future work

---

This study successfully constructs a theoretical framework and assessment model for evaluating high-stakes language test preparation platforms in higher education through Conceptual Framework Analysis (CFA). The proposed framework integrates Technology Architecture, Pedagogy Effectiveness, and Institutional Governance dimensions, grounded in the DeLone and McLean model. The assessment model defines fifteen operational indicators, and the structural path model maps these inputs to net benefits. The utility, completeness, and robustness of the framework are formally validated via linear programming and Monte Carlo simulations. This model fills a key gap in the CALL evaluation literature and provides a rigorous decision-support tool for university adoption decisions.
